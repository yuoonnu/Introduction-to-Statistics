# 📊 통계처리입문 과제

R 언어를 활용하여 데이터를 정제하고, 통계학적 이론을 실제 헬스케어 및 사회적 도메인에 적용해 보는 과정을 담고 있습니다.

## 👤 Identity & Goals
- **Interest**: 생명과학과 IT의 융합
- **Objective**: 데이터 기반의 통계적 사고력을 바탕으로 복잡한 생체 데이터에서 유의미한 비즈니스 및 보건학적 인사이트를 도출합니다.

## 🛠 Tech Stack
- **Language**: `R`
- **Core Skills**: 시각화, 기술 통계, 추론 통계, 가설 검정

## 📂 Project Structure & Assignments

### 1. 데이터 시각화 및 비판적 분석 (A2~A3)
* **[A2] 뉴스 기사 속 그래프 분석**: 의학, IT, 환경 분야의 실제 기사를 바탕으로 데이터 왜곡 없는 시각화 기법의 적절성을 평가했습니다.
* **[A3] R 기반 공공 데이터 시각화**: 통계청 데이터를 활용하여 사망 원인(원그래프), 의료서비스 이용률(꺾은선), 제약 연구개발비(누적 막대)를 시각화했습니다.

### 2. 기술 통계 및 산포 분석 (A4)
* **[A4] 화학물질 배출량 중심과 산포 분석**: 
    * 전국 지역별 화학물질 배출량의 평균, 중앙값, 사분위수 분석.
    * 극단값에 의한 데이터 편향(Skewness) 해석 및 히스토그램 비교 분석.

### 3. 확률 분포의 도메인 적용 (A5~A6)
* **[A5] 실생활 속 이산확률분포 사례 연구**: 
    * **이항 분포**: 유전자 변이 발생 확률 모델링.
    * **초기하 분포**: 의약품 품질 검사의 비복원 추출 모델링.
    * **음이항 분포**: 임상시험 성공 기준 달성 시뮬레이션.
    * **푸아송 분포**: 미생물 군집 수 데이터와 이론값 비교 검증.
* **[A6] 정규분포 및 표준화 사례 분석**:
    * 헬스케어 데이터(혈압, BMI)의 정규성 검정 및 시각화 비교.
    * 화학 성분 농도 관리에서의 $Z$-score(표준화) 활용 방안 고찰.

### 4. 추론 통계 및 가설 검정 (A7~A10)
* **[A7] 구간추정 실습**: 표본 크기에 따른 신뢰구간 폭의 변화를 분석하고, 미세먼지 및 혈압 데이터를 통해 표준정규분포와 t-분포의 적용 차이를 확인했습니다.
* **[A8] 단일 집단 가설 검정**: 현대인의 평균 체온 데이터를 분석하여 기존 의학 표준과의 통계적 차이를 입증했습니다.
* **[A9] 두 집단 비교 검정**: 심장질환 유무에 따른 최대 심박수 차이 및 연령대별 당뇨병 유병률의 유의미한 차이를 검증했습니다.
* **[A10] 분산 분석(ANOVA) 및 독립성 검정**:
    * 연령대(20~40대) 상승에 따른 수축기 혈압의 유의미한 증가 경향 입증.
    * 흡연 여부와 고혈압 발병 간의 통계적 상관관계(독립성 위배) 확인.

## 💡 Key Experience
- **Bio-IT 도메인 전문성**: 유전자 변이, 임상시험, 질병 유병률 등 헬스케어 데이터를 통계적으로 정교하게 다루는 경험을 축적했습니다.
- **비판적 데이터 해석**: 데이터의 편향성이나 표본의 특성을 고려하여 최적의 통계 모델을 선택하는 의사결정 역량을 확보했습니다 .
